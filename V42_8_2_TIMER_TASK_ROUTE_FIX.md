# Damasio OS V42.8.2 — Timer + Return Task Fix

Focus: fix persisted timer bug, make return visits clickable, and make task assignment scalable.

- Old browser timer sessions are cleared by storage version migration.

- Admin can select worker/crew from a reusable worker list and choose the return date per task.
- Topbar version updated to V42.8.2.
