# Damasio OS V50.2.8

## Included in this consolidated release

- Permanent dark Admin sidebar treatment, including the lower navigation area.
- Customer Mobile feedback changed from stars to Like / Dislike.
- Dislike supports Feedback Only or a red-priority Task sent to Admin.
- Feedback form closes after submission and becomes a completed card.
- Optional tips: $5, $10, $20, or custom. No $2 option.
- Confirmation dialog clearly shows the selected tip and explains that no payment is processed until Damasio Pay is activated.
- Existing ERP and desktop flows remain unchanged.

## Validation

Run locally with the project-required Node 20 and pnpm 9:

```bash
pnpm install --frozen-lockfile
pnpm build
```
