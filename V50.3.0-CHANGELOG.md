# Damasio OS V50.3.0 — ERP Stabilization

## Fixed in this build
- Admin sidebar now uses one continuous structural background for the entire document height.
- Removed the desktop sticky/100vh combination that exposed a white column while scrolling.
- Customer mobile feedback form closes after submission.
- The previous “feedback completed” card is no longer kept on screen.
- Existing Like/Dislike, Feedback Only/Create Task, and $5/$10/$20/Custom tip flow remains preserved.

## Roadmap preserved
- V50.x: ERP stabilization and integration.
- V51.0: Multi-company architecture, Master, company isolation, audit, temporary access, system/email codes, three access levels, Lead Center.
- V52: Finance with invoices, payments, expenses, payroll and reports using Stripe/Square.
