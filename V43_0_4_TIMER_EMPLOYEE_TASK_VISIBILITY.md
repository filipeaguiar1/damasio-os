# Damasio OS V43.0.4 — Timer Engine + Employee Task Visibility

- Storage version bumped to 43.0.4 to clear legacy browser service sessions.
- Old service timers from previous ZIP versions are removed on first load.
- Employee Service Issues now show only `assigned` and `in_progress` tasks.
- When Employee marks a return visit as `completed`, it disappears from Employee views.
- Completed tasks stay visible for Admin review and Customer history.
- Admin still controls final `Resolve`.
