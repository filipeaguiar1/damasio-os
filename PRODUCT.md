# Damasio OS — Product Vision

Damasio OS is a simple, premium SaaS platform for small exterior service businesses, starting with landscaping, lawn care and snow removal.

## Core philosophy

**Powerful to grow. Simple to use.**

The system should think first and let the user confirm. Every important action should happen in no more than three clicks when possible.

## Golden rules

1. No information should be entered twice.
2. Property is the center of the system.
3. Employee never sees financial data.
4. AI only suggests; Admin approves.
5. Every module must connect to the operational workflow.

## Main workflow

Request → Quote → Approved → Job → Route → Employee → Completed → Invoice → Payment → History

## Current product focus

V42 focuses on real database foundation through Supabase. The UI remains simple while the backend becomes real and scalable.
