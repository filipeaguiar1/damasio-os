# Damasio OS V51.2 — Operational Map

## Included
- Interactive Admin map with pan, zoom and selectable property markers.
- Status colours: Open blue, Done green, Skipped yellow, Task red, Available grey.
- Crew and status filters.
- Property detail panel with photo, service, crew, visit status and directions.
- No employee GPS, background tracking or live location.
- Optional latitude/longitude fields added to the canonical property record.

## Production note
The preview loads Leaflet and OpenStreetMap tiles from public endpoints. Before commercial high-volume use, configure a contracted tile provider such as Mapbox, Stadia Maps or another OpenStreetMap-compatible provider.
