# Damasio OS V50.3.1 — Integrated Reset + Skip

## Employee Mobile
- Added **Reset** to clear the active timer/session and return only that house to Open.
- Added **Skip** without forcing the employee to choose an exact reason.
- Skip accepts an optional comment and up to five optional photos.
- Skipped houses remain on the route with a yellow status.

## ERP integration
- Skip is stored in the shared service session record.
- Admin/Dispatch receive a notification.
- Activity log and workflow history record employee, property, timestamp, comment and photos.
- A skipped house is not marked Done and remains available for office review/rescheduling.
